# WoW-Bot
## Changing Database credentials :
Go to MySqlDatabaseHandler.cs and edit the following strings : 
- CharConnString
- WorldConnString
- AuthConnString

## Changing Token : 
Go to WoWBot.cs and edit the following string : 
- Token
